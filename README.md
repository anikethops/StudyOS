# StudyOS

StudyOS is a premium desktop productivity application for students and self-learners.

Built with:
- Electron
- React
- TypeScript
- Tailwind CSS
- Vite

## Current Progress

- Desktop app shell setup complete
- App routing and sidebar layout complete
- Dashboard UI foundation complete
- Preparing to build task management, focus timer, notes, progress tracking, and AI tools

## Planned Features

- Dashboard
- Task Management
- Focus Timer
- Notes System
- Progress Tracking
- AI Study Tools
- Smooth animations
- Selective Three.js visual experiences

## Goal

To create a smooth, useful, visually polished study and productivity OS for deep work and learning.

## Status

Currently in active development.
